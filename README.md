# Factorio RCON GUI
#### A GUI for Factorio's Dedicated Server RCON Console

### Current Features:
- Easy Server Connection-
- Subset of Commonly used commands including
    - Player Listing
    - Admin List
    - Server Save
    - Server Stop
- Features Still in progress
    - Player Kick / Ban
    -

  - [ ] Enable / Disable
